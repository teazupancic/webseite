CSS property kurze Erklärung möglichen Wertbelegungen ein Beispiel
color ändert die Farbe des Textes farben wie red,blue,lightblue... oder hexadecimal wie color:red
font-family legt eine andere Schriftart fest "web safe fonts": arial, Courier New, Georgia, Times New Roman,Trebuchet MS,Verdana family-font:arial
default fonts buchstaben haben einzelheiten oder nicht, style von Schrift, breite von Buchstaben serif (georgia), sans-serif(arial), monospace(Courier New), cursive, fantasy Unter ein Beispiel von font stacking
font-size font Grosse px(pixels),em,rem font-size:10px
font-style schräg text normal,italic(italic version von font),oblique(schräg normal version) font-style:italic
font-weight Legt fest, wie fett der Text ist, Fettdruck normal, bold, lighter, bolder, 100-900 font-weight:bold
text-transform transformiert text none, uppercase,lowecase,capital,full-width text-transform:uppercase
text-decoration dekoration none, underline, overline, line-trough text-decoration:underline
text-shadow shatten auf text 4 einzelheiten: horizontal offset, vertial offset, blur radius, base colour text-shadow: 4px 4px 5px black
text-align wo ist das Text right,left, center, justify (makes text spred out so that all lines are the same width) text-align:center
line-hight Höhe jeder Zeile nummer line-height:1.5 & line-height:0.5
letter-spacing & word-spacing den Abstand zwischen den Buchstaben und Wörter einstellen nummer of pixels letter-spacing: 2px word-spacing 4px
